# DOCKER

Notes for using Docker

```
docker image ls
#-- to see what images are available

docker ps
#-- to see what containers are running

docker compose ls
#-- to see what services are running

docker exec
#-- to execute a command within the docker

docker exec -it <container-name> /bin/sh
#ex: docker exec -it trmnl_byos_laravel-app-1 /bin/sh
#-- to open the shell (/bin/bash if bash setup)
```
