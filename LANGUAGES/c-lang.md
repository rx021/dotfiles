# C PROGRAMMING LANGUAGE

## SETUP

How to check if you have it on your system (linux):

```bash
g++ --version
```

## START

How to create a simple file:

```test.c
#include <stdio.h>

int main()
{
  printf("Hello! Bonjour mon ami");

  return 0;
}

```

How to compile the code:

```bash
cc -o executableFile test.c
```

How to run the code:

```bash
./executableFile
```
